# Green Earth

**Green Earth** is a responsive web project built with **Tailwind CSS** and **Vanilla JavaScript**, designed to showcase interactive UI elements, dynamic behavior, and clean, modern design. This project demonstrates practical frontend development skills and is beginner-friendly.

## 🌟 Features
- Fully **responsive layout** for mobile and desktop  
- **Interactive elements** using Vanilla JavaScript  
- **Clean and modern design** with Tailwind CSS  
- Beginner-friendly and easy to understand  
- Smooth user experience across devices  

## 💻 Technology Stack
- **HTML5** – Web page structure  
- **CSS3 / Tailwind CSS** – Styling and responsive design  
- **JavaScript (Vanilla JS)** – Dynamic behavior and interactivity
